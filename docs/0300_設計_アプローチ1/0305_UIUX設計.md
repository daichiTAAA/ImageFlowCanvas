# 1. レスポンシブデザイン

## 1.1. ブレークポイント

| デバイス | 幅             | レイアウト            | 検査機能対応                 |
| -------- | -------------- | --------------------- | ---------------------------- |
| Mobile   | < 768px        | 単一カラム、簡略表示  | モバイル検査アプリ専用UI     |
| Tablet   | 768px - 1024px | 2カラム、タッチ最適化 | 検査監視、タブレット検査対応 |
| Desktop  | > 1024px       | 3カラム、フル機能     | 管理機能、分析ダッシュボード |

## 1.2. アクセシビリティ

- WCAG 2.1 AA準拠
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 高コントラストモード
- **現場作業環境対応**：
  - 手袋着用時の操作性
  - 明るい環境での視認性
  - 大きなタッチターゲット
  - 音声フィードバック対応
